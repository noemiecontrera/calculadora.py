# Calculadora com Tkinter

Uma calculadora desenvolvida em Python utilizando a biblioteca gráfica *Tkinter*. O projeto foi criado para consolidar conhecimentos de interface de usuário (UI) e lógica de programação em Python.

Atualmente, o projeto realiza operações matemáticas básicas, com um planejamento estruturado para evoluir para uma calculadora científica completa.

---

## Funcionalidades Atuais

*   *Operações Básicas:* Adição, Subtração, Multiplicação e Divisão.
*   *Interface Limpa:* Layout intuitivo inspirado em calculadoras padrão.
*   *Tratamento de Erros:* Prevenção de problemas comuns, como a divisão por zero.


## Tecnologias Utilizadas

*   *Python:* Linguagem base do projeto.
*   *Tkinter:* Biblioteca nativa para o desenvolvimento da interface gráfica.


## Roadmap de Evolução (Próximos Passos)

O objetivo principal é transformar este projeto em uma *Calculadora Científica*. As próximas implementações programadas são:

- [ ] *Funções Trigonométricas:* Seno ($sin$), Cosseno ($cos$) e Tangente ($tan$).
- [ ] *Operações Avançadas:* Potenciação ($x^y$), Radiciação ($\sqrt{x}$) e Fatorial ($n!$).
- [ ] *Constantes Matemáticas:* Inclusão de botões para Pi ($\pi$) e Euler ($e$).
- [ ] *Logaritmos:* Logaritmo natural ($ln$) e base 10 ($log$).
